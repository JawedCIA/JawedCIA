## Featured Projects

### **[KavachRT](https://kavachrt.com)**

Open-source AI red-team framework. An autonomous operator attacks your LLM or AI system, adapts as it finds weaknesses, and returns a reproducible security score with a go/no-go verdict, before you ship. Currently my most ambitious project.

Live: [kavachrt.com](https://kavachrt.com)

---

### **[Mizaan](https://github.com/JawedCIA/mizaan-eval)**

Multi-agent LLM validation framework with human-governed memory consolidation. Three-layer pipeline (primary scorer, auditor agent, human validator) with retrieval-augmented continuous learning. Sole-authored preprint published on Zenodo, June 2026.

Paper: [DOI 10.5281/zenodo.21025602](https://doi.org/10.5281/zenodo.21025602)

---

### **[ArkThor](https://github.com/JawedCIA/ArkThor)**

Malware C2 traffic classification engine. Peer-reviewed research published at ICISSP 2024 in Rome, recognised as Best Position Paper Candidate.

Paper: [DOI 10.5220/0012420200003648](https://doi.org/10.5220/0012420200003648)
